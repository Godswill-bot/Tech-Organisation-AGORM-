# AGORM Website

A premium, animated startup website for AGORM built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

AGORM is a technology-driven organization that builds robust web and mobile applications while delivering innovative digital solutions across industries.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Features

- Modern hero with animated gradient/parallax background
- Scroll-triggered reveal animations across sections
- Sticky navbar with blur and smooth anchor navigation
- Services cards with hover interactions
- Portfolio grid with interactive project detail modal
- Team section with profile cards and social/contact links
- Contact section with responsive form and business contact details
- Responsive layout optimized for mobile, tablet, and desktop
- SEO-friendly metadata and clean semantic structure

## Project Structure

- src/app: Next.js app layout, page, and global styles
- src/components/site: Reusable website sections/components
- src/data/site-content.ts: Centralized content model for services, projects, team, and navigation

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Production Build

```bash
npm run build
npm start
```
